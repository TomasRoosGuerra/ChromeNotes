# Chrome Notes - React Web App

A modern, feature-rich note-taking web application built with React, TypeScript, and Firebase.

---

## ğŸ“– Description

Chrome Notes is a hierarchical note-taking app designed for organizing thoughts, tasks, and information efficiently. It features a two-level tab system (main tabs with sub-tabs) and a rich text editor with markdown support.

### Key Highlights

- **Hierarchical Organization** - Main tabs with sub-tabs for structured note-taking
- **Rich Text Editor** - TipTap-powered editor with full formatting capabilities
- **Task Management** - Built-in checkboxes with completion tracking
- **Cloud Sync** - Real-time synchronization with Firebase Firestore
- **Offline Support** - Works offline, syncs automatically when online
- **Import/Export** - Copy notes to markdown, import from clipboard
- **Dark Mode** - Auto-detect system preference
- **Mobile Responsive** - Beautiful UI on all devices
